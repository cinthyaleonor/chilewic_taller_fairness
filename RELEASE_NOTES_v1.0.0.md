# Workshop Materials - ChileWiC 2024

**Release v1.0.0 - Initial Release for Zenodo Publication**

**Taller de Ética y Fairness en Inteligencia Artificial: Un Caso de Sesgo de Selección**

Presented at **ChileWiC 2024** - XII Encuentro de Mujeres en Computación  
Date: November 29, 2024  
Location: Universidad Mayor, Santiago, Chile

## 📦 What's Included

This release contains all materials from the workshop "IA, Ética y Fairness: Un Caso de Sesgo de Selección" presented at ChileWiC 2024.

### New Features

- ✅ Complete workshop materials ready for Zenodo publication
- ✅ Citation metadata (CITATION.cff and .zenodo.json)
- ✅ MIT License for open access
- ✅ Comprehensive README with academic structure
- ✅ Python code for fairness analysis and bias mitigation
- ✅ Presentation materials in Markdown format
- ✅ Requirements file for easy environment setup

### Key Components

1. **Fairness Analysis Code**: Python scripts for analyzing and mitigating bias in machine learning models
2. **Case Study**: Simulated hiring data with gender bias for practical analysis
3. **Presentation**: Complete workshop presentation in Markdown and PDF formats
4. **Documentation**: Comprehensive README with installation and usage instructions
